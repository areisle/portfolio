Planning of the State Machine

In planning out this project, one of the first considerations was what I was going to use a state machine for, and then carefully planning out what the states would be and how their transitions would occur. I wanted something very visual for my example so that it would be easy to see how the machine was behaving and what it was accomplishing. I choose to do a bird because of the large spectrum of visual dichotomous movements they\'re capable of. Figure 2 shows the original plan for the states of the bird.

Due to time constraints, the number of states had to be scaled back. Figure 2 shows the states currently implemented.
